# barbershop

Цель работы: синхронизация процессов и управление потоками, изучить
проблему координации взаимодействия параллельных процессов в условиях
ограниченных ресурсов и методы её решения средствами операционных
систем.
Условие: представьте себе небольшую парикмахерскую, состоящую из одной
комнаты с одним рабочим местом (парикмахером) и несколькими стульями
ожидания для клиентов. Парикмахерская работает следующим образом:
- Когда клиенты приходят, они садятся на свободный стул ожидания, пока
парикмахер занят другим клиентом.
- Если все места заняты, новый клиент уходит.
- Когда работа выполнена, парикмахер зовёт следующего клиента.
- Если новых клиентов нет, парикмахер засыпает и ждёт появления нового
посетителя.
Напишите программу, моделирующую взаимодействие между клиентами и
парикмахером. Используйте потоки или процессы для моделирования
действий клиентов и парикмахера. Реализуйте правильную систему
блокировки и сигнализации (использование мьютексов, семафоров, условных
переменных и других механизмов синхронизации) для предотвращения
конфликтов доступа к ресурсам (рабочему месту и местам ожидания).
Программа должна выдавать отчёт о происходящем процессе (клиент пришёл
/ ушёл, ждет в зале, обслуживается).
